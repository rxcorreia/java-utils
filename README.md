# java-utils

A collection of utility classes for Java projects, covering the following data structures: String, Date/time, Collections, XML,
File, Email, etc...
Extensive use of existing Java, Apache and Joda time libraries.

## Requirements

* Java Development Kit (JDK) 21 or newer
* Maven 3.9+

## Building

```bash
mvn clean verify
```

Code coverage reports are generated with JaCoCo at `target/site/jacoco/index.html`.
